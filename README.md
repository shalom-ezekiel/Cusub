# Cusub

Cusub is a subscription-tracking platform that helps individuals and teams monitor, optimize, and manage their subscriptions in one place. It provides insights, detects unused or redundant subscriptions, and sends alerts to reduce unnecessary spend.

## Features

- Dashboard for tracking subscriptions and usage  
- Alerts for upcoming renewals and unusual activity  
- Detects unused or redundant subscriptions  
- Analytics and usage trends for better decision-making  

> ⚠️ Tech and integrations are continuously evolving. Check updates for the latest improvements.

## Tech Stack (Current)

- **Frontend:** React, TypeScript, Tailwind CSS  
- **Backend:** FastAPI / Node.js (under continuous improvement)  
- **Database:** PostgreSQL (core storage, may evolve to other solutions)  
- **Real-time / Cache:** Redis / WebSockets (optional modules)
